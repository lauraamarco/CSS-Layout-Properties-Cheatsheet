Display

The display property is an important CSS property for controlling layout. It specifies whether an HTML element is treated as a block or an inline element. Every HTML element has a default display value, depending on what type of element it is. The default display value for most elements is block or inline. The display property is used to change the default display behavior of HTML elements.

A block-level element ALWAYS starts on a new line and takes up the full width available. An example for elements that have a block display by default are <div>, <h1> - <h6>, <p>, <form>, <header>, <footer>, <section>

An inline element DOES NOT start on a new line and only takes up as much width as necessary. These elements will show with inline display by default: <span>, <a>, <strong>, <em>, <img>

Codecademy Docs - Display
Property Value Description
Display block These are rendered as block-level elements. They create a line break after the element and occupy the full width of their parent container.
inline Elements with display: inline; are rendered as inline elements. They don’t create line breaks and occupy only the necessary width and height based on their content.
inline-block Elements with display: inline-block; are rendered as inline-level block elements. They behave like inline elements but allow you to set width, height, padding, and margins.
none Elements with display: none; are not rendered and are completely hidden from the page. They don’t take up any space in the document flow. It is commonly used with JavaScript to hide or show elements without deleting and recreating them.
flex Elements with display: flex; create a flex container, allowing you to build flexible and responsive layouts. Flex items inside the container can be positioned and resized using flexbox properties.
grid Elements with display: grid; create a grid container that enables you to define rows and columns, creating a two-dimensional grid system. Child elements inside the container can be placed in specific cells and adjusted for size and alignment.
